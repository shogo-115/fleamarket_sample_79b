# DB設計
## usersテーブル
|Column|Type|Option|
|------|----|------|
|nick_name|string|null: false|
|email|string|null: false, unique: true|
|password|string|null: false|
### Association
- has_many :products
- has_many :comments
- has_one :names, dependent: :destroy
- has_one :address, dependent: :destroy
